These files go in specific places. For my setup:

The transforms go in:
```
vissl-env/lib/python3.7/site-packages/vissl/data/ssl_transforms/
```

The data source goes in:
```angular2html
vissl-env/lib/python3.7/site-packages/vissl/data/
```

The .yaml configuration file goes in:
```angular2html
vissl-env/lib/python3.7/site-packages/configs/config/pretrain/simclr/
```
